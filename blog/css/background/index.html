Background

O background é uma propriedade css que configura o plano de fundo (background) de um elemento. O background pode ser uma cor, uma imagem, um degradê ou a junção de todos esses.

Background Color

Para adicionar cor de fundo utilizamos a propriedade background ou background-color e valor da cor que desejamos. Veja mais sobre cores.

div { background: #affc41 } div { background-color: #affc41 }

Background Image

Image

Quando se dejesa adicionar imagem ou degradê (gradiente) utiliza-se o background-image.

div { background: url(image.jpg) } div {
 background-image: url(image.jpg)
}

Resultado:

Gradient

Já para adicionar um degradê é necessário informar como valor a função gradient.

div {
 background-image: linear-gradient
 (180deg, #b2ff9e, #affc41, #1dd3b0)
}
